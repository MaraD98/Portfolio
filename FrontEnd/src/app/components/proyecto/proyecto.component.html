Encriptador

Codificador de texto con las siguientes caracteristicas y condiciones:

  • Tecnologías utilizadas: CSS / JS / HTML
  • Debe funcionar solo con letras minúsculas
  • No deben ser utilizados letras con acentos ni caracteres especiales
  • Debe ser posible convertir una palabra para la versión encriptada y para la version desencriptada.
  • La pagina debe estar construida visualmente usando HTML y CSS.
    • Ejemplo de encriptación: "gato" => "gaitober" gaitober" => "gato".
